# Farah Tanveer | Computer Science Portfolio

A premium, ATS-optimized personal portfolio showcasing my journey as a Computer Science student at the University of the Punjab (2024-2028). 

This platform highlights my exploration into the vastness of computer science, specifically focusing on **Systems Development**, **Information Security**, and **AI/ML**.

## 🚀 Key Features
- **Real-time GitHub Integration**: Dynamic metrics pulling live data from GitHub to showcase active growth and contribution.
- **Glassmorphism Design**: A modern, high-end visual aesthetic with fluid transitions and responsive layouts.
- **Project Showcases**: Detailed case studies of technical projects, including engineering reasoning and outcomes.
- **Technical Competencies**: A structured overview of my proficiency in languages (C++, Python, SQL), frameworks (Flask), and developer tools (Linux, Git, Figma).

## 🛠️ Technology Stack
- **Backend**: Flask (Python)
- **Frontend**: Vanilla HTML5, CSS3, JavaScript
- **Deployment**: Configured for Render/Heroku with Gunicorn
- **Icons**: DevIcon & FontAwesome

## 📂 Project Highlights
- **Netra**: A network monitoring system for real-time packet tracking and analysis.
- **SecurePass**: A security-focused tool for analyzing and validating password strength.
- **UrbanRide**: A structured relational database system modeled for digital car rental services.

## 🏁 Deployment
This repository is pre-configured for deployment on **Render**.
1. Connect this GitHub repository to your Render account.
2. Select the **Python** environment.
3. The build and start commands are automatically handled by the `Procfile` and `requirements.txt`.
